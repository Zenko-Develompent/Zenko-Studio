# Условные операторы в Bash

В Bash ветвление делается через `if`, `elif`, `else`.

## Пример

```bash
if (( n > 0 )); then
  echo "YES"
else
  echo "NO"
fi
```

## Сравнения

Для чисел удобно использовать `(( ... ))`:

- `>` , `<` , `>=` , `<=`
- `==` , `!=`

Для строк часто используют `[ ... ]` или `[[ ... ]]`.

## Цепочка условий

```bash
if (( score >= 90 )); then
  echo "5"
elif (( score >= 70 )); then
  echo "4"
else
  echo "3"
fi
```

## Типичные ошибки

- Пропуск `then` или `fi`.
- Неправильный тип сравнения (строковый вместо числового).
- Отсутствие проверки граничных значений.

Главная идея урока: чётко формулируй условия и тестируй крайние случаи.
