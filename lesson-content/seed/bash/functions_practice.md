# Функции в Bash: практика

Практика по функциям помогает упростить сложные скрипты.

## Пример

```bash
max2() {
  local a=$1
  local b=$2
  if (( a > b )); then
    echo "$a"
  else
    echo "$b"
  fi
}

read x y
max2 "$x" "$y"
```

## Советы

- Используй `local` внутри функции.
- Возвращай вычисление через `echo` и захватывай `$(...)` при необходимости.
- Делай функции маленькими и понятными.

Главная цель: разбивать задачу на переиспользуемые части.
