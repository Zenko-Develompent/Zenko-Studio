# Переменные в Bash: практика

Практический шаблон почти всегда одинаковый: прочитать вход, сохранить в переменные, вычислить ответ, вывести его.

## Базовый пример

```bash
#!/usr/bin/env bash
read a b
echo $((a + b))
```

## Что важно

- В `read` порядок переменных важен.
- Арифметика делается через `$(( ... ))`.
- Не добавляй лишний текст в вывод.

## Самопроверка

- Обработал ли я все входные значения?
- Совпадает ли формат вывода с условием?
- Нет ли лишних пробелов или строк?

Главная цель: стабильно решать задачи «ввод -> вычисление -> вывод».
