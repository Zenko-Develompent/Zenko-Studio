# Массивы и строки в Bash

Bash поддерживает массивы и базовые строковые операции.

## Массивы

```bash
arr=(1 2 3 4)
echo "${arr[0]}"    # 1
echo "${#arr[@]}"   # длина массива
```

Обход массива:

```bash
sum=0
for x in "${arr[@]}"; do
  ((sum += x))
done
echo "$sum"
```

## Строки

```bash
s="HELLO"
echo "${#s}"      # длина
echo "${s:0:1}"   # первый символ
echo "${s: -1}"   # последний символ
```

## Частые ошибки

- Пропуск кавычек вокруг переменных.
- Неверный синтаксис срезов строки.
- Путаница между `${arr[@]}` и `${arr[*]}`.

Главная идея урока: аккуратная работа с кавычками и индексами особенно важна в Bash.
