# Массивы и строки в Bash: практика

Здесь тренируем обход массивов и базовую обработку строк.

## Пример: сумма массива

```bash
read n
read -a arr
sum=0
for x in "${arr[@]}"; do
  ((sum += x))
done
echo "$sum"
```

## Пример: первый и последний символ

```bash
read s
first="${s:0:1}"
last="${s: -1}"
echo "$first $last"
```

## Что проверить

- Есть ли кавычки вокруг строковых переменных.
- Не выходим ли за границы строки.
- Совпадает ли формат вывода с условием.

Главная цель: безошибочно работать с индексами и строками в Bash.
