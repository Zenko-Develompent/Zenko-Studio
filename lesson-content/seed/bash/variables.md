# Переменные в Bash

В Bash переменные обычно строковые, но часто используются и как числа в арифметике.

## Объявление

```bash
name="Аня"
age=12
```

Важно: вокруг `=` не ставятся пробелы.

## Получение значения

```bash
echo "$name"
```

Для чтения из stdin:

```bash
read a b
echo $((a + b))
```

## Имена переменных

- Используй понятные имена: `total_score`, `student_age`.
- Допустимы буквы, цифры, `_`.
- Не начинай имя с цифры.

## Арифметика

```bash
x=5
x=$((x + 2))
echo "$x"  # 7
```

Главная идея урока: переменные хранят входные данные и промежуточные результаты, а аккуратный синтаксис Bash критичен к мелочам.
