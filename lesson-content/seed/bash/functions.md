# Функции в Bash

Функции в Bash помогают структурировать скрипт и убирать повтор кода.

## Объявление функции

```bash
sum2() {
  local a=$1
  local b=$2
  echo $((a + b))
}
```

Вызов:

```bash
result=$(sum2 7 8)
echo "$result"
```

## Аргументы

- `$1`, `$2`, ... — позиционные аргументы.
- `local` ограничивает переменную внутри функции.

## Почему это полезно

- Скрипт легче читать.
- Одну функцию можно вызывать многократно.
- Упрощается отладка.

## Практика

Для задач хакатона часто достаточно функций: `max2`, `is_even`, `print_range`.

Главная идея урока: функции в Bash делают скрипты более управляемыми и понятными.
