# Циклы в Bash

Циклы в Bash позволяют повторять действия и обрабатывать набор данных.

## Цикл for

```bash
for i in {1..5}; do
  echo "$i"
done
```

## Цикл while

```bash
i=1
while (( i <= 5 )); do
  echo "$i"
  ((i++))
done
```

## Накопление суммы

```bash
sum=0
for i in {1..5}; do
  ((sum += i))
done
echo "$sum"
```

## Ошибки

- Не изменяется счетчик.
- Неправильная граница цикла.
- Лишние пробелы в арифметике и выражениях.

Главная идея урока: цикл + корректная граница + корректный шаг = правильный результат.
