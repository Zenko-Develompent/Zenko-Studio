# Циклы в Bash: практика

В этом уроке решаем задачи, где нужен повтор.

## Пример: вывести 1..N

```bash
read n
for ((i=1; i<=n; i++)); do
  echo "$i"
done
```

## Пример: сумма 1..N

```bash
read n
sum=0
for ((i=1; i<=n; i++)); do
  ((sum += i))
done
echo "$sum"
```

## Контроль качества

- Стартовая точка цикла верная.
- Верхняя граница выбрана правильно.
- Счетчик изменяется каждый шаг.

Главная цель: уверенно использовать циклы для числовых задач.
