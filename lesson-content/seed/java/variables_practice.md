# Переменные в Java: практика

В этом уроке фокус на задачах с вводом и вычислениями.

## Шаблон решения

1. Прочитать входные данные.
2. Сохранить их в переменные.
3. Выполнить вычисление.
4. Вывести результат в точном формате.

## Пример: сумма двух чисел

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();
        System.out.println(a + b);
    }
}
```

## На что обратить внимание

- Тип переменной должен соответствовать данным.
- Если нужен целый результат, выбирай `int` или `long`.
- Формат вывода должен совпадать с условием (переносы, пробелы).

## Самопроверка

- Понимаю ли я, какие данные приходят на вход?
- Все ли переменные названы понятно?
- Нет ли лишнего текста в выводе?

Главная цель: научиться быстро собирать корректный «ввод -> обработка -> вывод».
