# Массивы и строки в Java: практика

Здесь тренируем обход массивов и обработку строк.

## Массив: стандартный цикл

```java
int[] a = {1, 2, 3, 4};
int sum = 0;
for (int i = 0; i < a.length; i++) {
    sum += a[i];
}
System.out.println(sum);
```

## Строка: базовые операции

```java
String s = sc.next();
System.out.println(s.length());
System.out.println(s.charAt(0));
System.out.println(s.charAt(s.length() - 1));
```

## Частые задачи

- Длина строки.
- Первый/последний символ.
- Сумма элементов массива.
- Поиск максимума/минимума.

## Типичные ошибки

- `i <= a.length` вместо `i < a.length`.
- Попытка взять `charAt(length)`.
- Неправильный формат входа.

Главная цель: работать с индексами безопасно и предсказуемо.
