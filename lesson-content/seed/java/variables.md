# Переменные в Java

Переменная — это именованная область памяти, где хранится значение определённого типа. В Java тип указывается обязательно: `int`, `double`, `boolean`, `String` и т.д. Это помогает компилятору заранее проверять ошибки.

## Как объявлять переменные

```java
int age = 12;
double price = 99.5;
String name = "Аня";
boolean isOnline = true;
```

`=` — это присваивание. Справа вычисляется выражение, слева сохраняется результат.

## Правила и стиль

- Имя должно быть понятным: `studentAge`, `totalScore`.
- Нельзя начинать имя с цифры.
- Нельзя использовать пробелы.
- Лучше не делать слишком короткие имена вроде `x1`, если это не временная переменная.

## Изменение значения

```java
int x = 5;
x = x + 2; // теперь x равно 7
```

Переменная может менять значение много раз, пока программа выполняется.

## Ввод и вывод

В задачах часто нужно читать данные из ввода и печатать ответ. На уровне базовой логики важно понимать принцип: прочитали значение -> обработали -> вывели.

Главная идея урока: переменная хранит данные, а тип определяет, какие операции с ними допустимы.
