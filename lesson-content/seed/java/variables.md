# Переменные в Java

Переменная хранит значение по имени. Перед использованием переменную нужно объявить с типом.

Примеры типов: `int`, `double`, `String`, `boolean`.

```java
int age = 12;
String name = "Аня";
System.out.println(age);
```

Совет: выбирайте понятные имена переменных, чтобы код было легче читать.
