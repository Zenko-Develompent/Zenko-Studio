# Условные операторы в Java: практика

Здесь тренируем задачи, где результат зависит от условия.

## Универсальный алгоритм

1. Прочитать входные данные.
2. Сформулировать точное условие.
3. Записать ветви `if/else`.
4. Проверить на граничных примерах.

## Пример: четное или нечетное

```java
int n = sc.nextInt();
if (n % 2 == 0) {
    System.out.println("EVEN");
} else {
    System.out.println("ODD");
}
```

## Полезные шаблоны

- `if (n > 0)` — положительное число.
- `if (n >= l && n <= r)` — попадание в диапазон.
- `if (...) ... else if (...) ... else ...` — несколько категорий.

## Проверки перед сдачей

- `n = 0`
- минимальные/максимальные значения из условия
- отрицательные значения

Главная цель: писать ветвления без логических дыр и конфликтов условий.
