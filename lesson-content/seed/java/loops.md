# Циклы в Java

Циклы нужны, когда одно и то же действие нужно выполнить много раз. В Java чаще всего используют `for` и `while`.

## Цикл for

Подходит, когда заранее известен диапазон повторений.

```java
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
```

Структура: инициализация -> условие -> шаг.

## Цикл while

Подходит, когда количество шагов заранее неизвестно.

```java
int i = 1;
while (i <= 5) {
    System.out.println(i);
    i++;
}
```

Важно менять переменную в условии, иначе получится бесконечный цикл.

## Накопление результата

Типичный шаблон:

```java
int sum = 0;
for (int i = 1; i <= n; i++) {
    sum += i;
}
System.out.println(sum);
```

## Частые ошибки

- Неверные границы (`i < n` вместо `i <= n`).
- Сброс накопителя внутри цикла.
- Отсутствие изменения счётчика.

Главная идея урока: цикл + правильные границы = корректное решение множества задач.
