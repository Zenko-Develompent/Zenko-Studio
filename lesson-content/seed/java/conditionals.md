# Условные операторы в Java

Условные операторы позволяют программе принимать решения. Базовая конструкция — `if`, а для альтернативной ветки используется `else`.

## Базовый синтаксис

```java
if (n > 0) {
    System.out.println("YES");
} else {
    System.out.println("NO");
}
```

В круглых скобках должно быть логическое выражение (`true`/`false`).

## Сравнения и логика

Часто используются операторы:

- `>` , `<` , `>=` , `<=`
- `==` (равно)
- `!=` (не равно)
- `&&` (логическое И)
- `||` (логическое ИЛИ)

Пример составного условия:

```java
if (age >= 7 && age <= 15) {
    System.out.println("Школьник");
}
```

## Цепочка проверок

```java
if (score >= 90) {
    System.out.println("5");
} else if (score >= 70) {
    System.out.println("4");
} else {
    System.out.println("3");
}
```

Порядок `if/else if` важен: выполняется первая подходящая ветка.

## Частые ошибки

- Путать `=` и `==`.
- Неправильно расставлять границы (`<` вместо `<=`).
- Не проверять граничные значения (0, 1, отрицательные).

Главная идея урока: условие должно быть точным и проверяться в правильном порядке.
