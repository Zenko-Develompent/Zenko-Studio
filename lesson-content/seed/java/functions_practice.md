# Функции в Java: практика

Практика функций — это разбиение задачи на небольшие независимые блоки.

## Мини-шаблон

```java
static int max2(int a, int b) {
    return a > b ? a : b;
}
```

В `main`:

```java
int a = sc.nextInt();
int b = sc.nextInt();
System.out.println(max2(a, b));
```

## Подход к решению

1. Найти повторяющуюся логику.
2. Вынести в метод.
3. Передавать только нужные параметры.
4. Вернуть только нужный результат.

## Что проверять

- Метод не зависит от «скрытых» переменных.
- Тип возвращаемого значения совпадает с задачей.
- Название метода отражает смысл (`sum`, `isEven`, `max2`).

Главная цель: писать компактные и переиспользуемые решения.
