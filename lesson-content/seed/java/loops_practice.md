# Циклы в Java: практика

Практика по циклам — это работа с повторяющимися действиями и аккуратными границами.

## Шаблоны

### Вывод диапазона

```java
for (int i = 1; i <= n; i++) {
    System.out.println(i);
}
```

### Накопление суммы

```java
int sum = 0;
for (int i = 1; i <= n; i++) {
    sum += i;
}
System.out.println(sum);
```

### Цикл с while

```java
int i = 1;
while (i <= n) {
    // действие
    i++;
}
```

## Типичные баги

- Неправильная стартовая точка (`i = 0` вместо `1`).
- Ошибка на последнем шаге (`<` вместо `<=`).
- Неизменяемый счетчик.

## Рекомендация

Сначала выпиши первые 3-4 шага цикла вручную, потом кодируй.

Главная цель: уверенно считать, выводить и фильтровать данные в циклах.
