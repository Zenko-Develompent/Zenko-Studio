# Массивы и строки в Java

Массив хранит набор элементов одного типа. Строка (`String`) хранит текст.

## Массивы

```java
int[] a = {1, 2, 3, 4};
System.out.println(a[0]); // 1
```

- Индексация начинается с `0`.
- Длина массива: `a.length`.

Обход массива:

```java
int sum = 0;
for (int i = 0; i < a.length; i++) {
    sum += a[i];
}
```

## Строки

```java
String s = "HELLO";
System.out.println(s.length()); // 5
System.out.println(s.charAt(0)); // H
```

Полезные операции:

- `length()` — длина строки.
- `charAt(i)` — символ по индексу.
- `toLowerCase()` / `toUpperCase()` — изменение регистра.

## Частые ошибки

- Выход за границы массива/строки.
- Неправильные границы цикла (`<=` вместо `<`).
- Игнорирование пустого ввода.

Главная идея урока: аккуратная работа с индексами и длиной решает большинство задач по массивам и строкам.
