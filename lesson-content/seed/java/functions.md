# Функции в Java

Функция (метод) — это блок кода с именем, который выполняет конкретную задачу. Она помогает убрать дублирование и сделать код понятнее.

## Пример метода

```java
static int square(int x) {
    return x * x;
}
```

Вызов:

```java
int result = square(7); // 49
```

## Что важно понимать

- Параметры — входные данные метода.
- `return` — возвращаемое значение.
- Тип перед именем (`int`, `String`, `void`) показывает, что метод возвращает.

## Почему это полезно

- Один алгоритм можно использовать много раз.
- Легче тестировать маленькие части программы.
- Проще читать и изменять код.

## Метод без возврата

```java
static void printHello(String name) {
    System.out.println("Привет, " + name);
}
```

`void` означает, что метод ничего не возвращает.

## Практический подход

Сначала разбей задачу на шаги, затем вынеси каждый повторяющийся шаг в отдельный метод.

Главная идея урока: методы делают программу структурной и поддерживаемой.
